Setup Instructions

  1. Run python scripts/import_chips.py to load chips + UNet++ predictions into the database.
  2. Share your server's IP address (e.g. http://192.168.x.x:8000) with labelers.
  3. Labelers open that URL in their browser, register with their name, and save their token.
  4. Use the "Assign Chips" panel above to distribute chips among labelers.
  5. Monitor progress in "Labeler Progress" — refresh as needed.
  6. When labeling is complete, export COCO JSON for downstream model training/evaluation.

Backup: The entire database is a single file — iceberg_labels.db. Copy it to OneDrive regularly as a backup.
