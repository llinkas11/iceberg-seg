Labeling Guide ▾

Label as Iceberg: The visible above-water portion of the iceberg body.

Do NOT label:

  • Open water
  • Shoreline or land
  • Dark regions that are not icebergs
  • Sea ice or melange (tag the chip instead)

Common confusions:

  • Iceberg vs. sea ice: If not a distinct iceberg, do not label.
  • Dark patches: Only label if it is clearly an iceberg.

Before submitting, check:

  • Labeled only iceberg bodies?
  • Avoided open water / dark patches?
  • Corrected obvious model mistakes?
  • Added missing iceberg regions?
  • Tagged ambiguous chips?
